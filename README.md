# 도서 관리 API 제작 Side Project

## 1. 개요
로컬 환경에서만 관리되던 도서 관리 프로그램은 로컬 PC가 고장나면 데이터의 유실 가능성이 있고, 보안에도 취약하다.
이를 개선하기 위해, 클라우드 기반의 '도서 관리 프로그램'을 제작한다.
